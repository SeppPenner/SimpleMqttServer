SimpleMqttServer
====================================

SimpleMqttServer is a project to run a simple [MQTT server](https://github.com/chkr1011/MQTTnet) from a json config file. The project was written and tested in .Net 5.0.

```json

  "Port": 1883,
  "Users": [
    {
      "UserName": "Hans",
      "Password": "Test"
    }
  ]

```
